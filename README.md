# Digital Image Processing and Computer Vision

> **SE494 – Digital Image Processing**  
> Chiang Mai University

---

## Course Description

This course introduces the fundamental concepts and practical techniques of **Digital Image Processing (DIP)** and **Computer Vision (CV)**. Students learn how digital images are acquired, processed, analyzed, and interpreted using both classical image processing algorithms and modern deep learning approaches.

The course emphasizes hands-on implementation using **Python**, **OpenCV**, and deep learning frameworks, allowing students to develop real-world computer vision applications.

---

## Course Learning Outcomes (CLOs)

After completing this course, students will be able to:

1. Explain the principles of digital image processing and computer vision.
2. Apply appropriate image processing techniques to solve practical problems.
3. Develop computer vision applications using OpenCV and AI-based methods.

---

## Course Topics

### Part I — Digital Image Processing

- Introduction to Digital Images
- Image Acquisition
- Image Representation
- Sampling and Quantization
- Image Enhancement
- Histogram Processing
- Spatial Filtering
- Frequency Domain Processing
- Image Restoration
- Morphological Image Processing
- Image Segmentation
- Feature Extraction

---

### Part II — Computer Vision

- Image Classification
- Object Detection
- Semantic Segmentation
- Instance Segmentation
- Pose Estimation
- Object Tracking
- Video Processing
- Deep Learning for Vision
- Transfer Learning
- Vision Transformers (Introduction)
